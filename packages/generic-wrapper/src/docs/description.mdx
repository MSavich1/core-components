### Лейаут

Компонент представляет из себя структуру с тремя вертикально или горизонтально расположенными слотами. Каждый слот может принять в себя любой компонент или кастомную верстку. Вертикальные и горизонтальные отступы можно настраивать независимо друг от друга.

```jsx live
render(() => {
    const [state, setState] = React.useState({
        direction: 'horizontal',
    });

    const [direction, setCalendarPosition] = React.useState('horizontal');

    const addon = (
        <div
            style={{
                background: 'rgba(55, 120, 251, 0.1)',
                border: '1px dashed #3778FB',
                borderRadius: '8px',
                height: '64px',
                width: '64px',
                boxSizing: 'border-box',
            }}
        />
    );

    return (
        <Container>
            <Row>
                <div style={{ backgroundColor: 'rgba(55, 120, 251, 0.1)', padding: '16px' }}>
                    <GenericWrapper direction={direction}>
                        <GenericWrapper.Addon>{addon}</GenericWrapper.Addon>
                        <Gap size='l' direction={direction} />
                        <GenericWrapper.Addon>{addon}</GenericWrapper.Addon>
                        <Gap size='l' direction={direction} />
                        <GenericWrapper.Addon>{addon}</GenericWrapper.Addon>
                    </GenericWrapper>
                </div>
            </Row>
            <div style={{ marginTop: '20px' }}>
                <RadioGroup
                    label='Направление'
                    direction='vertical'
                    name='radioGroupFirst'
                    onChange={(_, { value }) => setCalendarPosition(value)}
                    value={direction}
                >
                    <Radio label='Горизонтальное' value='horizontal' />
                    <Radio label='Вертикальное' value='vertical' />
                </RadioGroup>
            </div>
        </Container>
    );
});
```

### GenericWrapper.DataContent

Компонент из четырех вертикальных слотов и может использоваться только в качестве контента слота GenericWrapper.Addon. Каждый из слотов может иметь левую и правую части. Отступы внутри компонента могут быть настроены вручную.

```jsx live
render(() => {
    const Addon = ({ width = 320 }) => (
        <div
            style={{
                background: 'rgba(55, 120, 251, 0.1)',
                border: '1px dashed #3778FB',
                borderRadius: '8px',
                height: '20px',
                width: `${width}px`,
                boxSizing: 'border-box',
            }}
        />
    );
    return (
        <Container>
            <GenericWrapper direction='vertical'>
                <GenericWrapper.Addon>
                    <GenericWrapper.DataContent>
                        <GenericWrapper.DataContent.Addon
                            leftSide={{ content: <Addon width={208} /> }}

                        />
                        <Gap direction='vertical' size='3xs' />
```
